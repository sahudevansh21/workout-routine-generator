# 💪 Workout Routine Generator

## Problem
Many individuals struggle to create diverse and effective workout routines, especially when exercising at home or with limited equipment. They often repeat the same exercises, leading to boredom or plateaus, and finding new ideas without subscription apps can be challenging.

## Solution
This website allows users to generate custom workout routines based on their available equipment, target muscle groups, and desired duration. It provides a library of exercises and can save generated routines locally, helping users stay motivated and vary their training without external dependencies.

## Getting Started

### Prerequisites
Make sure you have Node.js (v18 or later) and npm installed.

### Installation

1.  **Clone the repository (or create the files from the JSON output):**
    ```bash
    # If creating from JSON, copy all files into their respective paths.
    # Otherwise, if it were a git repo:
    # git clone <repository-url>
    # cd workout-routine-generator
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

### Running the Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Building for Production

```bash
npm run build
```

This command builds the application for production deployment.

### Starting the Production Server

```bash
npm run start
```

After building, this command starts a Next.js production server.
